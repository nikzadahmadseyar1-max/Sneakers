🛒 shopping Website Project
📌 Overview

This is a modern and responsive shopping website built using HTML, CSS, and JavaScript.
The project includes essential features required for an online store such as product listing, shopping cart, and user interaction.

🚀 Features
🛍️ Product listing page
🔎 Search and filter functionality
🛒 Shopping cart system
👤 User-friendly interface
📱 Fully responsive design (mobile, tablet, desktop)
⚡ Fast and optimized performance

🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
